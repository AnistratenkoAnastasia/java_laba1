# Laba Container: Custom Container

## Singly Linked List Implementation

Лабораторная работа по реализации пользовательской структуры данных в Java. Проект демонстрирует работу с односвязным списком без использования встроенных коллекций, а также применение Maven, JUnit и JavaDoc.

## Возможности

- Контейнер для хранения целых чисел на основе односвязного списка (без использования встроенных коллекций)
- Добавление элементов в начало списка
- Получение элемента по порядковому номеру
- Удаление по значению (первое/последнее/все вхождения)
- Удаление по индексу
- Проверка на пустоту
- Очистка контейнера
- Полное покрытие кода комментариями Javadoc
- Unit-тесты на JUnit 5
- Автоматическая генерация документации через Maven

## Требования

- JDK 17+
- Maven (встроенный wrapper `mvnw.cmd`)
